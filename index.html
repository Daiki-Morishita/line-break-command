SubLineについて

SubLineは、動画の字幕・テロップ作りに特化した日本語自動改行ツールです。Googleが開発したオープンソースの分かち書きエンジンBudouXを搭載し、文の意味の切れ目を判定して、指定した最大文字数内で読みやすく改行します。

主な機能

  • 最大文字数を指定して、AI（BudouX）が意味の切れ目で自動改行
  • 句読点を文字数から除外するオプション
  • 句読点削除・空白行削除・句読点を半角スペースに置換
  • 「」『』を " " ' ' に一括変換
  • 文字の一括置換
  • .txt / .csv / .srt / .vtt / .json への書き出し
  • 文字数ルーラー表示で視認性◎

こんな方におすすめ

  • YouTube・TikTok・Instagram Reels の動画編集者
  • Premiere Pro / Final Cut Pro / DaVinci Resolve でテロップを作る方
  • セミナー・講演動画に字幕を入れる制作者
  • 長文を読みやすく整形したいライター・編集者

使い方

  1. 左側のテキストエリアに原稿を貼り付け
  2. 最大文字数を設定（字幕は20〜30文字が一般的）
  3. 右側に自動改行された結果が表示されます
  4. 「コピー」または「書き出し」で結果を取得

完全無料・インストール不要・会員登録不要。すべての処理はブラウザ内で完結し、入力したテキストはサーバーに送信されません。
